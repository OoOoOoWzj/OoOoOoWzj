### GitHub 统计卡片
将这行代码复制到你的 markdown 文件中，就是如此简单！
``` 
[![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra)](https://github.com/anuraghazra/github-readme-stats)
``` 
更改 ?username= 的值为你的 GitHub 用户名。

### 热门语言卡片

>热门语言卡片显示了 GitHub 用户常用的编程语言

将此代码复制粘贴到您的 README.md 文件中，并修改链接。
``` 
[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra)](https://github.com/anuraghazra/github-readme-stats)
``` 
端点: api/top-langs?username=anuraghazra

[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra)](https://github.com/anuraghazra/github-readme-stats)

## 显示编码时长
> **wakatime** 记录coding时间的工具

设置步骤如下：
- 注册 wakatime 账号
- 在自己常用的 IDE 上下载 wakatime 插件，配置上自己的 API key
- 将自己的 API key 存到自己 GitHub 仓库的 secrets
- 在 README 文件上加上
  ```
  <!--START_SECTION:waka-->
  <!--END_SECTION:waka-->
  ```
- 配置 GitHub 仓库的 Action，这样就能每天自动运行了

具体说明可参考 github 上的[athul/waka-readme项目](https://github.com/athul/waka-readme)
